                                Install Flutter #

                                • Install flutter and check flutter doctor
                                • Install Idea & Android Studio with flutter & dart plugins
                                • sudo apt install libstdc++-12-dev

                                Other Dependencies #

                                • sudo apt-get install -y postgresql-client or if on Fedora sudo dnf install postgresql
                                • sudo dnf install nodejs (fedora)

                                Prepare database & redis images #

                                • Install git & login to bitbucket to clone signage & tuxkit projects
                                • git config --global user.email "srix55@gmail.com" git config --global user.name "Srikanth Venkatesh"
                                • sudo apt install podman buildah
                                • cd ~/Devspace/espace/signage/devops/database
                                • ./generate-dockerfile.sh -t dev-1.0 & follow the instructions to run signagedb container
                                • podman run -p 6379:6379 --name redis docker.io/library/redis

                                Prepare the server #

                                • Install sdkman (curl -s "https://get.sdkman.io" | bash) & install java (sdk install java) & gradle (sdk install gradle)
                                • Cd into signage-server and run gradle wrapper
                                • sudo npm i -g @redocly/cli@latest
                                • cd ~/Devspace/espace/signage/signage-server/api/openapi && openapi lint specs/openapi.yaml && openapi bundle --output dist/openapi.yaml specs/openapi.yaml && cd ../.. && ./gradlew build -x test

                                Prepare the client #

                                • sudo apt install libstdc++-12-dev or sudo dnf install gcc libstdc++-devel
                                • sudo apt install libmpv-dev mpv or sudo dnf install mpv mpv-devel
                                • sudo apt install libsqlite3-dev or sudo dnf install sqlite sqlite-devel

                                Run #

                                • podman start signagedb redis
                                • cd ~/Devspace/espace/signage/signage-server && ./gradlew :signage-server-quarkus:quarkusDev
                                • Set config.dart installationType to desktop_lin
                                • Check api address in rest_client/api.dart
                                • cd ~/Devspace/espace/signage/client/admin/signage && flutter run -d linux

                                Google Cloud #

                                • Install gcloud & do gcloud init to set defaults
                                • Check out devops/webserver/build_deploy_webserver.sh
