                                Project Creation & Services #

                                • Create the GCP project - signage, preferably with srix55 google account
                                • Enable the Google Artifact Registry API
                                  • Create a new docker repository - gcloud artifacts repositories create docker-repo --repository-format=docker --location=asia-southeast1 --description="Docker repository"
                                • Enable the Compute Engine API
                                • Configure local gcloud
                                  • gcloud config set project "long-parity-395505" (the project id in quotes)
                                • Create a static ip address
                                  • gcloud compute addresses create ip-web --description=webserver\ static\ ip\ address --region=asia-southeast1
                                • Verify gcloud config by gcloud config list

                                Webserver Image #

                                • Remove unwanted images & containers
                                  • podman ps -a for containers
                                  • podman images for images
                                  • podman rm <container-id>
                                  • podman rmi <image-id>
                                • Devops -> web-server
                                • ./generate-dockerfile.sh -i webserver -t gcp-1.0
                                • buildah bud -f webserver-dockerfile -t webserver:gcp-1.0 .
